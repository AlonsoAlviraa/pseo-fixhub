Fixing Maytag Error F2/E1

Official troubleshooting guide for Maytag Oven (Gemini Series)

Diagnostic Summary

Severity High
Est. Cost $50 - $110

How to Fix It

  1. 1

    Turn off the oven and unplug it.

  2. 2

    Check the oven temperature sensor.

  3. 3

    Test the sensor resistance (should be ~1080 ohms at room temp).

  4. 4

    Replace the temperature sensor if faulty.
